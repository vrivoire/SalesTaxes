Le projet est à l'URL suivate: https://github.com/vrivoire/SalesTaxes

Dans le répertoire du projet cloné (SalesTaxes):

La structure de la base de données SQL est dans le fichier "tables.sql"
Le panier peux etre changé dans le fichier salesTaxes.properties

Construire le projet:
mvn clean install

Lancer le projet:
	java -jar target/SalesTaxes-1.0-SNAPSHOT.jar
